Requirements:

1) Have the python version 3.10.8 
2) Do the installation for the dotenv (pip install python-dotenv), (conda install -c conda-forge python-dotenv)

You have to open the file Examples_Cred.env and put in the part after the = your host,port and password and it works.

if you have already installed redis and dotenv, you can update them the doing 
(pip install --upgrade redis), (conda install --upgrade redis), (pip install --upgrade python-dotenv), conda install --upgrade python-dotenv)
